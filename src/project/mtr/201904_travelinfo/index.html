This mini website is for users traveling from Hong Kong to China on the High-Speed Rail (HSR). Its main purpose is to introduce China's famous tourist hotspots and travel plans on the map.

Since Google Maps is not available in China, I had to find a platform that could also be used in Hong Kong and China. Instead of Google Maps, amap (高德地圖) was used, which is a very influential online map platform in China.

The MTR website has the same character as the government, it must comply with Web Accessibility. I follow the guidelines of the W3C Various web accessibility functions such as font adjustment and tab key movement were applied.

Key Features

  • amap(高德地圖) API
  • Outside CMS export
  • Data handle by JSON
  • Web Content Accessibility according to W3C guidelines.
